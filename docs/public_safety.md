# Public Repository Safety

This repository is public. Treat it as source code and redacted fixtures only.

Do not commit:

- API keys
- tokens
- local absolute paths
- biometric traces
- private model weights
- private benchmark cases
- raw personal logs
- private conversation exports
- unredacted run manifests
- proprietary third-party content

Safe to commit:

- schemas
- deterministic source code
- redacted example reports
- public-safe candidate manifests
- documentation
- tests
