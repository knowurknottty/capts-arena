# Inversion Arena

CAPT delta benchmarking and recursive self-improvement system.

## Mission

The Inversion Arena measures architecture-level uplift over fixed basis.
It compares organism states under identical benchmark conditions,
detects real improvement versus cosmetic fluency,
preserves failures as reusable training artifacts,
and promotes only changes that improve measured behavior
without violating provenance, contradiction, freshness, or trust constraints.

## Sequence

```text
organism_state_A vs organism_state_B
    ↓
existing benchmark runner (capt-context or equivalent)
    ↓
pairwise comparator
    ↓
delta scorer
    ↓
kingboard
    ↓
failure museum
    ↓
promotion gate
```
